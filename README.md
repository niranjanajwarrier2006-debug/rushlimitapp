# Rush Limit - Proactive Transit Network

A smart transit management system for Trivandrum city that uses ML predictions to optimize bus deployments.

## Features
- Passenger check-in system
- Bus owner dashboard with ML insights
- Real-time bus tracking
- Demand prediction algorithms
- Optimized bus deployment
